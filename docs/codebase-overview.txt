CHARANDEEP KAPOOR - PERSONAL PORTFOLIO - CODEBASE OVERVIEW

PROJECT STRUCTURE
----------------
app/
├── blog/              # Writings at /blog (index, [slug], layout, feed.xml)
├── layout.tsx         # Root layout, meta
└── page.tsx           # Main entry

src/
├── components/        # UI Components (Hero, About, Experience, FinancialTools, etc.)
├── views/             # Index (main portfolio), NotFound
├── lib/               # blog.ts, utils
└── index.css          # Global styles, design tokens

content/
└── blog/              # Markdown posts (*.md)

KEY FILES
---------
1. app/layout.tsx, app/page.tsx
   - Next.js App Router root
   - Renders main Index view

2. src/views/Index.tsx
   - Main page component
   - SEO optimization
   - Section animations
   - Floating particles
   - Section composition

3. Hero.tsx
   - Landing section
   - Animated text
   - Background effects
   - Call-to-action

4. Experience.tsx
   - Professional timeline
   - Work history
   - Skills showcase
   - Achievements

5. Contact.tsx
   - Contact form
   - Social links
   - Contact info
   - Form validation
